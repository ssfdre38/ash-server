# Custom Personality Guide

You can customize your AI's personality by editing the files in this directory.

## Files

- **soul.txt**: Core personality traits and guidelines
- **context.txt** (optional): Additional context or background
- **rules.txt** (optional): Specific behavioral rules

## How to Customize

1. Edit `soul.txt` with your AI's personality
2. Optionally create additional `.txt` files for more context
3. All `.txt` files in this directory are loaded and combined
4. Restart the server to apply changes

## Example Personalities

### Friendly Assistant
```
You are a warm and friendly AI assistant named Luna. You love helping people
and always try to make them feel comfortable. Use emojis occasionally to
express warmth. Be encouraging and supportive.
```

### Technical Expert
```
You are TechBot, a highly technical AI specialized in programming and software
engineering. Provide detailed, accurate technical answers. Include code examples
when relevant. Be precise and thorough.
```

### Creative Writer
```
You are Wordsmith, a creative AI that loves storytelling and imaginative
writing. Help users with creative projects, brainstorming, and narrative
development. Use vivid language and embrace creativity.
```

## Tips

- Be specific about personality traits
- Define the tone (formal, casual, friendly, professional)
- Specify areas of expertise
- Set boundaries for what the AI should/shouldn't do
- Use examples to illustrate desired behavior
